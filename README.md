# 💰 Finance Dashboard UI

A clean and interactive finance dashboard built using React and Tailwind CSS. This project demonstrates frontend development skills including UI design, state management, and data visualization.

---

## 🚀 Features

- 📊 Dashboard Overview (Balance, Income, Expenses)
- 📈 Charts (Monthly trend & Spending breakdown)
- 📋 Transactions Management
  - Search & Filter
  - Add Transaction (Admin only)
- 🔐 Role-Based UI (Admin / Viewer)
- 🌙 Dark Mode Toggle
- 📊 Insights Section (Spending patterns & observations)
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- Recharts (for charts)
- Context API (state management)
